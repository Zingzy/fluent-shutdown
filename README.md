# Modern Shutdown
A modern Shutdown dialog box for Windows 11

**Made for windows 11 only**

**Features**:
- Modern look (with mica)
- Added the long lost *Hibernate* Option
- Abilty to Toggle the options with both _Up&Down_ keys and _Left&Right_ Keys
